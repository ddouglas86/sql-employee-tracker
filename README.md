  ## Description
  ### A Content Management System created to manage a company's employee database using Node.js, Inquirer, and MySQL. The user is able to view and manage all departments, roles, and employee information within the database.

  ## Installation Instructions
  ### Clone the repo to your local machine. Open the integrated terminal in VSCode. Run 'npm install'. Once dependencies are loaded, use 'mysql' to open a new session. Run 'source ./db/schema.sql' then 'source ./db/seeds.sql' to create and seed the database.

  ## Usage
  ### Open the integrated terminal once installed and use command 'node index.js'. Once connected, follow the prompts on screen to manage the database.
